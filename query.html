QUERY function

● Truy vấn dữ liệu trên một bảng cố định dựa vào câu lệnh SQL.

 VD1: tìm tất cả các chuỗi con bắt đầu với PERF (phần tô màu).

 VD2: Triển khai bảng thống kê ở dạng PIVOT (truy vấn không hỗ trợ keyword PIVOT).

Support functions:

- DAY/MONTH/YEAR - DATEADD/DATEPART/DATEDIFF/TIMESTAMP - ISINT/ISNUMBER/ISDATE - AVERAGE/RAND/SIGN/ROUNDUP/ROUNDDOWN/ROUNDNEAREST - CONCAT/JOIN/TEXTBEFORE/TEXTAFTER/LEFT/MID/RIGHT/SPLIT_PART - FIND/FINDOF/FINDLAST/FINDLASTOF/MATCHES/COUNTW - FORMAT/PROPER/SENTENCE/REPT/REVERSE/ERASE - GETINT/GETINTS/GETNUM/GETALPHA/GETALPHANUM

Related function

XQUERY   Truy vấn dữ liệu giữa nhiều bảng dựa vào câu lệnh SQL.

PIVOT   Tương tự chức năng PIVOT trong Power Query.
