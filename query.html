QUERY function

● Truy vấn dữ liệu trên một bảng cố định dựa vào câu lệnh SQL.

 VD1: tìm tất cả các chuỗi con bắt đầu với PERF (phần tô màu).

 VD2: Triển khai bảng thống kê ở dạng PIVOT (truy vấn không hỗ trợ keyword PIVOT).

Support functions:

 1. Các hàm DATETIME:

   - HOUR/MINUTE/SECOND/DAY/MONTH/YEAR

   - DATEADD/DATEPART/DATEDIFF/TIMESTAMP

 2. Các hàm LOGIC:

   - ISINT/ISNUMBER/ISDATE

   - ISUPPER/ISLOWER/ISALPHA/ISDIGIT/ISALPHANUM/ISSYMBOL/ISLIKE

   - ISBLANK/ISZERO/ISNULL

   - GREATER/SMALLER/BETWEEN/EQUAL/ISODD/ISEVEN

 3. Các hàm CONVERT:

   - TO.INT/TO.NUM/TO.TEXT/TO.ENG

   - ROUNDUP/ROUNDDOWN/ROUNDNEAREST

   - UPPER/LOWER/PROPER/SENTENCE

 4. Các hàm MATH:

   - AVERAGE/PLUS/MINUS/PRODUCT/DIVIDE/RAND//SUMIF/MEDIAN

   - SUM.A/MAX.A/MIN.A/AVERAGE.A

   - SQRT.A/CEILING.A/TRUNC.A/EXP.A/FLOOR.A/MOD.A/POW.A/POWER.A/SIGN.A/ABS.A/

 5. Các hàm ARRAY:

   - INDEX/MATCH.C/SORT/FIRST/LAST/TAKE/DROP/HSTACK

   - FILTER/NONBLANK/NONZERO/UNIQUE

 6. Các hàm STRING:

   - FIND/FINDOF/FINDLAST/FINDLASTOF

   - LEFT/MID/MID.W/MID.AT/RIGHT

   - CONCAT/JOIN/TEXTBEFORE/TEXTAFTER/SPLIT.PART/SPLIT

   - FORMAT/REPT/REVERSE/ERASE

   - TEXT.DIGIT/TEXT.INT/TEXT.NUM/TEXT.ALPHA/TEXT.ALPHANUM

   - REG.SPLIT/REG.EXTRACT/REG.REPLACE/REG.MATCH/REG.COUNT

 7. Các hàm STATISTICAL:

   - COUNTA/COUNTBLANK/COUNT.A/COUNTZERO/COUNTNULL

   - COUNTIF/COUNT.W

Related function

XQUERY   Truy vấn dữ liệu giữa nhiều bảng dựa vào câu lệnh SQL.

PIVOT   Tương tự chức năng PIVOT trong Power Query.
