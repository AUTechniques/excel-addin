MAIL MERGE feature

● Trộn dữ liệu từ Excel sang Word mở rộng.

 Ảnh thiết lập hộp thoại.

Tính năng:

 1.0 Mặc định các trường dữ liệu thông thường (Number, Text,..) không cần tiền tố (vd: Ngày tháng, Họ Tên,..).

 2.0 Để trộn dữ liệu kèm Hypelink, đặt tên cột với tiền tố "H_"

 3.1 Để trộn dữ liệu kèm Ảnh, đặt tên cột với tiền tố "I_"

 3.2 Các trường dữ liệu ảnh chỉ cần chứa tên ảnh hoặc URL ảnh (vd: Logo.png, Avatar.jpg,..).

 4.0 Để trộn dữ liệu kèm bảng (Table) từ các file Word, đặt tên cột với tiền tố "T_"

 5.0 Các hình ảnh có thể đặt trong thư mục bất kỳ, tuy nhiên cần thêm địa chỉ các thư mục này vào mục [Path].

 6.0 Đảm bảo các dữ liệu hợp lệ (ví dụ tên gmail, tên ảnh, web link..)

 7.1 Để đính kèm ảnh trực tiếp vào File, tích chọn mục [Attach], sau đó bạn có thể gửi file trực tiếp cho người nhận mà không cần đính kèm ảnh.

 7.2 Nếu không [Attach file], bạn cần gửi file và cả thư mục ảnh đính kèm cho người nhận.

 8.0 Mặc định mỗi hàng xuất ra một file, để nối tất cả các file với nhau, tích chọn mục [Join Files].

 9.0 Mặc định tất cả các hàng được xuất, nếu muốn xuất trên một vùng nhất định nhập vị trí dòng [From] to [To] .

 Xem video Demo.

Lưu ý:

 1. Mặc định, cột cuối cùng được sử dụng để đặt tên File Word được xuất (Tên file không chứa đuôi extension).

 2. Nếu gộp File, tên File đầu tiên sẽ được lựa chọn để xuất ra.

 3. Thư mục xuất ra nên được để trống, vì dữ liệu phía trong sẽ luôn được xóa và xuất mỗi khi chạy chương trình.

Related function

NUMBERTOWORDS (feature)   Thiết lập phụ từ cho ứng dụng đọc số thành chữ (Ribbon).

FLOOKUP    Nhập liệu nâng cao, hỗ trợ tra cứu (lọc) và xuất nhanh dữ liệu dựa trên từ khóa.

XGROUP   Hàm mở rộng của GROUP. (hỗ trợ các hàm nâng cao).
