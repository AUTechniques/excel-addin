QUICK ACCESS FOLDERS feature

● Truy cập, quản lý các file hoặc thư mục làm việc trực tiếp từ Excel hiệu quả, dễ dàng và nhanh chóng.

 Sử dụng hộp thoại để thiết lập.

Các chế độ:

 1. Mở nhanh file hoặc thư mục bằng cách double click vào label.

 2. Đổi tên file hoặc thư mục bằng cách sửa tên label.

 3. Di chuyển file hoặc thư mục bằng cách sử dụng trái chuột (left mouse) kéo thả label từ vị trí này sang vị trí khác.

 3. Copy file hoặc thư mục bằng cách sử dụng phải chuột (right mouse) kéo thả label từ vị trí này sang vị trí khác.

 4. Tự động đồng bộ hóa Files, Folders với cửa sổ Explorer.

 Xem demo Video.

Lưu ý:

 1. Nên chọn thư mục cụ thể (tránh chọn toàn bộ ổ đĩa C:\ hay D:\ sẽ mất nhiều thời gian hơn để tải).

Related function

NUMBERTOWORDS (feature)   Thiết lập phụ từ cho ứng dụng đọc số thành chữ (Ribbon).

FLOOKUP    Nhập liệu nâng cao, hỗ trợ tra cứu (lọc) và xuất nhanh dữ liệu dựa trên từ khóa.

XGROUP   Hàm mở rộng của GROUP. (hỗ trợ các hàm nâng cao).
