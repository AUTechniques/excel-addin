FILE function

● Trả về danh sách files trong đường dẫn chỉ định. (Hỗ trợ hypelinks, đệ quy)

 VD: Lấy danh sách files trong thư mục đồng thời tạo hypelinks.

Ghi chú:

 Đối với phiên bản [Premium], hàm FILES trả về tất cả các thông số bao gồm :

 1. Tên file (Name).

 2. Đường dẫn file (Path).

 3. Tên phần mở rộng (Extension).

 4. Ngày khởi tạo (Date Created).

 5. Ngày chỉnh sửa cuối cùng (Date Modified).

 6. Ngày truy cập cuối cùng (Date Accessed).

 7. Kích thước ảnh (Pixel).

 8. Kích thước file (Bytes).
